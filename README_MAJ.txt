MISE A JOUR SANS CONFLIT

1) Fermer l'application.
2) Dezipper ce ZIP : un dossier 'Gestion-Cuisine_STABLE' est cree.
3) Copier UNIQUEMENT votre dossier 'data' de l'ancienne version vers la nouvelle (optionnel).
   - Ancien: Gestion-Cuisine/data
   - Nouveau: Gestion-Cuisine_STABLE/data
4) Lancer via Lancer-Gestion-Cuisine.bat (Windows) ou lancer.sh (Linux/Mac).

Nouveautes:
- Exports PDJ persistants sauvegardes dans data/facturation_pdj/exports/YYYY-MM/
- Suppression des exports PDJ depuis l'interface.
- Lecture bons .xls/.xlsx/.xlsm, suppression lignes/factures, sauvegarde robuste.
